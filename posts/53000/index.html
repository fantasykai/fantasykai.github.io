推荐使用 Cmd Markdown 自带的各种图功能，详见 Cmd Markdown 高阶语法手册。

  • 例子：
  • 显示：

  • 例子：
  • 显示：

$$ \require{AMDcd} \begin{CD} A @>>> B @>{\text{very long label}}>> C \ @. @AAA @| \ D @= E @<<< F \ \end{CD} $$

在本例中，very long label 自动延长了它所在箭头以及对应箭头的长度，因而交换图表十分适合进行化学反应式的书写。

  • 显示：
