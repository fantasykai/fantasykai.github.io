е·ҘдҪңдёӯеёёз”ЁGit е‘Ҫд»ӨжҖ»з»“

gitж•ҙзҗҶзҡ„жңүеҘҪеҮ зҜҮдәҶпјҢдёҚиҝҮзЎ®е®һжҜ”иҫғйҮҚиҰҒпјҢеҸҲж”¶йӣҶдёҖдәӣжҜ”иҫғе®һз”Ёзҡ„

  • еҲҶж”Ҝж“ҚдҪң

  • жҡӮеӯҳж“ҚдҪң

  • еӣһйҖҖж“ҚдҪң

  • ж Үзӯҫж“ҚдҪң

  • еёёи§„ж“ҚдҪң

    • gitеҲӣе»әйЎ№зӣ®д»“еә“
    • еҝҪз•Ҙе·ІеҠ е…ҘеҲ°зүҲжң¬еә“дёӯзҡ„ж–Үд»¶
    • еҸ–ж¶ҲеҝҪз•Ҙж–Үд»¶
    • жӢүеҸ–гҖҒдёҠдј е…ҚеҜҶз Ғ

еҲҶж”Ҝж“ҚдҪң

  1. git branch еҲӣе»әеҲҶж”Ҝ
  2. git branch -b еҲӣе»әе№¶еҲҮжҚўеҲ°ж–°е»әзҡ„еҲҶж”ҜдёҠ
  3. git checkout еҲҮжҚўеҲҶж”Ҝ
  4. git branch жҹҘзңӢеҲҶж”ҜеҲ—иЎЁ
  5. git branch -v жҹҘзңӢжүҖжңүеҲҶж”Ҝзҡ„жңҖеҗҺдёҖж¬Ўж“ҚдҪң
  6. git branch -vv жҹҘзңӢеҪ“еүҚеҲҶж”Ҝ
  7. git brabch -b еҲҶж”ҜеҗҚ origin/еҲҶж”ҜеҗҚ еҲӣе»әиҝңзЁӢеҲҶж”ҜеҲ°жң¬ең°
  8. git branch вҖ“merged жҹҘзңӢеҲ«зҡ„еҲҶж”Ҝе’ҢеҪ“еүҚеҲҶж”ҜеҗҲе№¶иҝҮзҡ„еҲҶж”Ҝ
  9. git branch вҖ“no-merged жҹҘзңӢжңӘдёҺеҪ“еүҚеҲҶж”ҜеҗҲе№¶зҡ„еҲҶж”Ҝ
  10. git branch -d еҲҶж”ҜеҗҚ еҲ йҷӨжң¬ең°еҲҶж”Ҝ
  11. git branch -D еҲҶж”ҜеҗҚ ејәиЎҢеҲ йҷӨеҲҶж”Ҝ
  12. git branch origin :еҲҶж”ҜеҗҚ еҲ йҷӨиҝңеӨ„д»“еә“еҲҶж”Ҝ
  13. git merge еҲҶж”ҜеҗҚ еҗҲе№¶еҲҶж”ҜеҲ°еҪ“еүҚеҲҶж”ҜдёҠ

жҡӮеӯҳж“ҚдҪң

  1. git stash жҡӮеӯҳеҪ“еүҚдҝ®ж”№
  2. git stash apply жҒўеӨҚжңҖиҝ‘зҡ„дёҖж¬ЎжҡӮеӯҳ
  3. git stash pop жҒўеӨҚжҡӮеӯҳе№¶еҲ йҷӨжҡӮеӯҳи®°еҪ•
  4. git stash list жҹҘзңӢжҡӮеӯҳеҲ—иЎЁ
  5. git stash drop жҡӮеӯҳеҗҚ(дҫӢпјҡstash@{0}) з§»йҷӨжҹҗж¬ЎжҡӮеӯҳ
  6. git stash clear жё…йҷӨжҡӮеӯҳ

еӣһйҖҖж“ҚдҪң

  1. git reset вҖ“hard HEAD^ еӣһйҖҖеҲ°дёҠдёҖдёӘзүҲжң¬
  2. git reset вҖ“hard ahdhs1(commit_id) еӣһйҖҖеҲ°жҹҗдёӘзүҲжң¬
  3. git checkout вҖ“ fileж’Өй”Җдҝ®ж”№зҡ„ж–Үд»¶(еҰӮжһңж–Үд»¶еҠ е…ҘеҲ°дәҶжҡӮеӯҳеҢәпјҢеҲҷеӣһйҖҖеҲ°жҡӮеӯҳеҢәзҡ„пјҢеҰӮжһңж–Үд»¶еҠ е…ҘеҲ°дәҶзүҲжң¬еә“пјҢеҲҷиҝҳеҺҹиҮіеҠ е…ҘзүҲжң¬еә“д№ӢеҗҺзҡ„зҠ¶жҖҒ)
  4. git reset HEAD file ж’ӨеӣһжҡӮеӯҳеҢәзҡ„ж–Үд»¶дҝ®ж”№еҲ°е·ҘдҪңеҢә

ж Үзӯҫж“ҚдҪң

  1. git tag ж ҮзӯҫеҗҚ ж·»еҠ ж Үзӯҫ(й»ҳи®ӨеҜ№еҪ“еүҚзүҲжң¬)
  2. git tag ж ҮзӯҫеҗҚ commit_id еҜ№жҹҗдёҖжҸҗдәӨи®°еҪ•жү“ж Үзӯҫ
  3. git tag -a ж ҮзӯҫеҗҚ -m вҖҳжҸҸиҝ°вҖҷ еҲӣе»әж–°ж Үзӯҫе№¶еўһеҠ еӨҮжіЁ
  4. git tag еҲ—еҮәжүҖжңүж ҮзӯҫеҲ—иЎЁ
  5. git show ж ҮзӯҫеҗҚ жҹҘзңӢж ҮзӯҫдҝЎжҒҜ
  6. git tag -d ж ҮзӯҫеҗҚ еҲ йҷӨжң¬ең°ж Үзӯҫ
  7. git push origin ж ҮзӯҫеҗҚ жҺЁйҖҒж ҮзӯҫеҲ°иҝңзЁӢд»“еә“
  8. git push origin вҖ“tags жҺЁйҖҒжүҖжңүж ҮзӯҫеҲ°иҝңзЁӢд»“еә“
  9. git push origin :refs/tags/ж ҮзӯҫеҗҚ д»ҺиҝңзЁӢд»“еә“дёӯеҲ йҷӨж Үзӯҫ

еёёи§„ж“ҚдҪң

  1. git push origin test жҺЁйҖҒжң¬ең°еҲҶж”ҜеҲ°иҝңзЁӢд»“еә“
  2. git rm -r вҖ“cached ж–Үд»¶/ж–Үд»¶еӨ№еҗҚеӯ— еҸ–ж¶Ҳж–Үд»¶иў«зүҲжң¬жҺ§еҲ¶
  3. git reflog иҺ·еҸ–жү§иЎҢиҝҮзҡ„е‘Ҫд»Ө
  4. git log вҖ“graph жҹҘзңӢеҲҶж”ҜеҗҲе№¶еӣҫ
  5. git merge вҖ“no-ff -m вҖҳеҗҲе№¶жҸҸиҝ°вҖҷ еҲҶж”ҜеҗҚ дёҚдҪҝз”ЁFast forwardж–№ејҸеҗҲе№¶пјҢйҮҮз”Ёиҝҷз§Қж–№ејҸеҗҲе№¶еҸҜд»ҘзңӢеҲ°еҗҲе№¶и®°еҪ•
  6. git check-ignore -v ж–Үд»¶еҗҚ жҹҘзңӢеҝҪз•Ҙи§„еҲҷ
  7. git add -f ж–Үд»¶еҗҚ ејәеҲ¶е°Ҷж–Үд»¶жҸҗдәӨ

gitеҲӣе»әйЎ№зӣ®д»“еә“

  1. git init еҲқе§ӢеҢ–
  2. git remote add origin url е…іиҒ”иҝңзЁӢд»“еә“
  3. git pull
  4. git fetch иҺ·еҸ–иҝңзЁӢд»“еә“дёӯжүҖжңүзҡ„еҲҶж”ҜеҲ°жң¬ең°

еҝҪз•Ҙе·ІеҠ е…ҘеҲ°зүҲжң¬еә“дёӯзҡ„ж–Үд»¶

  1. git update-index вҖ“assume-unchanged file еҝҪз•ҘеҚ•дёӘж–Үд»¶
  2. git rm -r вҖ“cached ж–Үд»¶/ж–Үд»¶еӨ№еҗҚеӯ— (. еҝҪз•Ҙе…ЁйғЁж–Үд»¶)
