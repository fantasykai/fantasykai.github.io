安装 centOS 报错 No Caching mode page found

2017-01-18
  • 测试

简述

在阿里的镜像库上下载了centos7.2的iso文件，制作好U盘引导盘，在PC上安装时， 报错

No Caching mode page found Assuming drive cache:write through
省略一大堆。。。
/dev/root does not exist

解决办法

  • 重启后，在进入安装界面时，按 e 键

    修改
vmlinuz initrd=initrd.img inst.stage2=hd:LABEL=CentOS\x207\x20x86_64 rd.live.check quiet

  • 将hd:后面的内容替换为挂在的U盘路径即可

    如：
vmlinuz initrd=initrd.img inst.stage2=hd:/dev/sdb1:/ quiet

  • 注意： sdb1 即为U盘的挂在名称。硬盘一般是sda

  • 可以通过 ls /dev/sd* 查看

修改好后，按下 CTRL + X 就OK了
