ssh 设置别名的方式访问服务器

2017-09-07
  • linux

在Windows上时，一般通过各种客户端(putty、SecureCRT等)连接到远程服务器，而切换到了Mac上，有了强大的终端，自然要好好用起来了。 一般在终端上通过 ssh 命令进行远程服务器登录。

远程登录服务器命令 ssh

  • ssh username@ip地址

例如： ssh root@192.168.11.2

but 服务端一般多台服务器，好几个ip地址，记住这么多ip地址好难，但是根据不同服务器的用途给他们起个别名，就方便多了，能想到自然能实现，这里简述下具体玩法~

给服务器起个别名

  1. 在 ~/.ssh/ 目录下，新建一个config（有的话，就不用建了噻）

  2. vim config 文件，进行编辑

  • Host 别名 起一个自己记得住的，比如开发环境 dev 测试环境 test ..
  • HostName 服务器的ip address
  • User 远程登录的用户名
  • IdentitiesOnly yes 反正这个就配置yes

ssh 别名 走起

配置好文件后，直接 键入 ssh dev 就玩起来了
