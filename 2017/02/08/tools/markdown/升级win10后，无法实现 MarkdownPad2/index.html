  • 笔记

问题描述

最新的win10系统，自带了ubuntu子系统，诱惑很大，于是把win7换成了win10，
But 升级win10后，部分软件不兼容，安装好经常使用MarkdownPad2后，直接打开一个.md文件，就开始报错

对文档进行编辑时，同样报错

解决办法

不甘心好用的工具无法使用， 百度之，并顺利解决，这里做个记录：

下载该程序 awesomium_v1.6.6_sdk_win.exe，安装后，即可正常使用

  • 注意： 低版本的win10，可能无法安装awesomium_v1.6.6_sdk_win，win10还是升级吧。希望win10能逐步稳定下来
