  • 技术小栈

问题简述

idea启动通过maven，启动tomcat时，执行clean命令也报错

  • 报错信息：

    Failed to clean project Failed to delete ... tomcat / log

无法删除，想到tomcat可能没有关闭。想到昨天idea被我强杀的进程。tomcat没杀掉，查看进程，果然tomcat进程还在，杀掉进程后，重新执行
maven 命令，正常了~
