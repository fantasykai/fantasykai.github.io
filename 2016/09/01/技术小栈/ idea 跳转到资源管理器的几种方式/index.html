简述

开发过程有时候会需要跳转到文件所在文件夹（资源管理器）中，之前使用eclipse时，有Open Explorer这样的插件，在文件上右键就可以使用,学习使用idea后，有时候也需要这样的功能，这里整理下我了解到的三种方式~

idea 跳转到资源管理器的方式

方式一：右键 Show in Explorer

  • 在project页面右键需要跳转的文件，选择右键菜单中的 Show in Explorer

方式二： 在工具栏上添加 Show in Explorer 按钮

  • 在工具栏的空白处，右键选择 Customize Menus and Toolbars…

  • 在对话框中，展开 Main Toolbar -> TasksToolbar -> 右侧的 Add After

  • 在对话框中，展开 Other -> Show in Explorer -> 为其设置一个图标
  • 注意：图标大小（建议18 * 18 / 16 * 16） 太大了，idea会提示icon太大

  • 更改图标：添加过后，可以在 Main Toolbar 最下面找到Show in Explorer 右边菜单选择 Edit Action Icon 进行更换图标

  • 这样比在project里右键文件省事一些~

方式三： 快捷键 ，idea强大的快捷键（推荐）

  • 直接在当前编辑的文件中，按下快捷键 Ctrl + Alt + F12 会列出文件的所在的文件夹及上层目录，回车，跳转吧
