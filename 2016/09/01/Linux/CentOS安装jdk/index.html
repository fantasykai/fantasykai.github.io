CentOS上安装JDK

2016-09-01
  • 技术小栈

简述

简单记录下Linux下安装JDK的方法，实际有很多种安装方式，这里只记录下解压缩包的方式，其他如，yum，rpm等方式不再陈述

准备环境

  • CentOS 7.0 64
  • jdk-8u74-linux-x64.tar.gz

安装过程

1.在/usr/local/目录下创建java目录

# mkdir /usr/local/java
# cd /usr/local/java

2.解压tar.gz 文件

# tar -zxvf jdk-8u74-linux-x64.tar.gz

3.设置环境变量

#  vi /etc/profile

  • 设置JAVA_HOME环境

    export JAVA_HOME=/usr/local/java/jdk1.8.0_74
export JRE_HOME=/usr/local/java/jdk1.8.0_74/jre
export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar:$JRE_HOME/lib:$CLASSPATH
export PATH=$JAVA_HOME/bin:$PATH

4.引用文件，生效

# source /etc/profile

5.验证配置

# java -version
java version "1.8.0_74"
Java(TM) SE Runtime Environment (build 1.8.0_74-b02)
Java HotSpot(TM) 64-Bit Server VM (build 25.74-b02, mixed mode)
