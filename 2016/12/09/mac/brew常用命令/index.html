brew常用命令

2016-12-09
  • 工具

mac上安装应用的命令，同apt-get，yum一样好使

安装

  • Homebrew安装命令，mac下自带ruby，在终端输入以下命令，按提示安装即可 

    ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go)"

  • 参考 Homebrew

brew常用命令：

安装软件:  brew install  ＊（软件名 eg. brew install nginx）

卸载软件:  brew uninstall ＊

搜索软件:  brew search *

显示已经安装软件列表:  brew list

更新软件:  brew update

更新某具体软件: brew upgrade git
